To get the calico outputs, follow below:

Unzip all_cases.zip
Add your solutions to current directory and rename them as convex.cpp and prim.cpp
Run the following Command in the shared Docker image
python -m calico.cli all_test_cases_ninova.t
Copy & save the terminal output as calico_output.txt
If you are applying for re-evaluation, do not forget to copy the outputs into your re-evaluation form!
